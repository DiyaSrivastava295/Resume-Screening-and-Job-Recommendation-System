# Resume Screening and Job Recommendation System

This is an in-progress machine learning project focused on resume screening and job recommendation using NLP techniques.

The goal of the project is to analyze resume text, extract relevant skills, and recommend suitable job roles based on similarity matching and text processing methods.

## Current Progress

* exploring NLP preprocessing techniques
* experimenting with TF-IDF vectorization
* studying similarity-based recommendation methods
* building the initial project pipeline

## Planned Features

* resume text preprocessing
* skill extraction from resumes
* job-role recommendation
* similarity scoring between resumes and job descriptions
* simple interactive interface

## Technologies Planned

* Python
* Pandas
* scikit-learn
* NLP preprocessing
* TF-IDF vectorization

## Project Status

Currently in the early development and research phase.

## Future Improvements

* improve recommendation accuracy
* support multiple resume formats
* build a GUI using Tkinter
* add resume-job matching scores
* experiment with advanced NLP models
